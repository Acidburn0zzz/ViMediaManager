v0.6α7

released on November 11, 2012
  • Dropped the unofficial IMDb and the Top250 API's in favour of a scraper that takes it's information from the IMDb itself.
  • Added an option to show poster images in the list. (Note that this may decrease performance)
  • Instead of completely refreshing the list, it now updates it. meaning it'll only add new movies if it finds them instead of adding every movie every time, and removes items it can no longer find.
  • Added recommendations to the metadata editor, but you'll have to update your movies first.
  • You can now update a single movie by right-clicking it in the list and choosing 'update movie'.
  • You can now get rid some of the visual clutter by disabling one or more of the mini-icons in the list.
  • Changed the layout of the main movie information display, showing more information with less clutter.
  • Updated growlnotify from 1.3 to 2.0.
  • Preliminary Boxee support option, which places database ID's at the bottom of the NFO, and adding full shell path for the local poster image thumb.
  • Added new options to sort the list by movie file size and duration, and added checkmarks to display which options are currently selected.
  • Added 'kijkwijzer' symbols to go with the new MPAA rating information.
  • Improved search results by adding a direct IMDb search back-up.
  • Introduced some caching to the search results window, and tweaked the layout a bit.
  • Made items that are marked as watched a bit more noticeable by marking their background.
  • The main & metadata editor's windows have gotten new toolbar icons to match the latest Mac OS X Style. (And are retina ready once there's a stable 'cocoa' build)
  • Made the actors list re-orderable.
  • Changed the way it searches for valid .nfo files.
  • Enabled drag-reordering in the sets manager, but be warned, it's not entirely stable.
  • First studio is set as default studio while the rest is placed aside, fixing this XBMC Skin bug: http://forum.xbmc.org/showthread.php?tid=82987
  • Fixed a bug where genres where only downloaded in english, now downloads in the correct localisation when available.
  • Fixed where it would incorrectly read the subtitle data, leaving it with nothing to write back down to the NFO file on a subsequent save.
  • Fixed a bug where it would incorrectly handle VIDEO_TS files and extra images during automatic renaming.
  • Re-added missing trailer icon in the metadata editor’s toolbar.
  • Fixed where special characters wouldn't show correctly in the search window.
  • Lots of bug fixes and changes.

I've put a lot of work in making sure that this time the release runs smoothly.

Here's to hoping that it does!

Enjoy!
